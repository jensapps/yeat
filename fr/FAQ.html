FAQ

1 / Comment fonctionne la Yeat?

cliquez ici (et référence à l'explication sur le site lui-même)

2 / Comment sont affichés les restaurants dans Yeat?

Il est important pour une utilisation optimale de l'application d'activer votre «fonction de localisation» dans votre téléphone.

Lorsque vous ouvrez l'application, vous obtenez immédiatement un aperçu de tous les restaurants qui ont une table disponible dans votre région. Si vous avez indiqué des favoris, vous ne pouvez également consulter que cette liste.

Si votre fonction de localisation est désactivée, vous pouvez uniquement consulter la liste des tableaux disponibles par ordre alphabétique.

3 / Comment gérer les notifications?

Réponse: Vous pouvez définir vos préférences de notification en fonction de vos besoins.

Vous pouvez gérer les notifications sur votre téléphone à tout moment. Ils annulent de toute façon les paramètres de l'application.Étant donné la nature de dernière minute de l'application, nous vous recommandons de laisser vos notifications pour Yeat afin de ne manquer aucune opportunité.

Voulez-vous obtenir une table pour l'un des prochains jours dans un restaurant généralement complet? marquez les restos souhaités comme vos favoris, assurez-vous que vos notifications sont activées dans votre téléphone.Yeat vous enverra alors des notifications de tous vos restaurants marqués comme favoris lorsqu'ils mettent une table en ligne.

4 / Dois-je payer quelque chose en tant qu'utilisateur pour utiliser Yeat?

L'application est entièrement gratuite pour l'utilisateur. L'utilisateur accepte les termes et conditions de Yeat en utilisant l'application. Lisez les conditions générales #lien ici.

2 / Que se passe-t-il si une table réservée n'apparaît pas?

Réponse: Les termes et conditions, approuvés par l'utilisateur, stipulent qu'un utilisateur qui a réservé la table s'engage à effectivement prendre la table avec le nombre de personnes disponibles à la date et à l'heure indiquées.Si l'utilisateur ne se présente pas sans aucune forme de communication, le restaurateur se réserve le droit de demander à l'utilisateur une compensation.Cette décision est prise par le propriétaire du restaurant lui-même et se fait directement entre l'utilisateur et le propriétaire du restaurant.

3 / combien de tables puis-je mettre en ligne en même temps.

Autant que vous le souhaitez.

4 / Combien de temps ma table restera-t-elle en ligne?

Lorsque vous mettez votre table en ligne, vous devez indiquer jusqu'à quelle heure vous souhaitez rendre la table visible. Lorsque la table est réservée, l'utilisateur sera informé que la table n'est plus disponible. Ce message reste visible pour une courte période.

5 / Puis-je mettre une table hors ligne?

Oui, vous pouvez mettre la table hors ligne à tout moment sans frais.

6 / Comment les utilisateurs peuvent-ils voir mon restaurant?

Lorsque vous enregistrez votre restaurant pour vous inscrire, vous remplissez toutes vos coordonnées et vous pouvez également télécharger quelques photos. Les utilisateurs peuvent ensuite vous trouver, soit par nom, emplacement ou d'autres filtres tels que le score Gault & Millau ou les étoiles Michelin.

7 / Je possède un autre restaurant. Dois-je créer un nouveau compte?

Oui. Un seul compte est possible par adresse professionnelle.

8 / Je veux qu'un employé gère Yeat. Que devrais-je faire ?

Envoyez-nous un e-mail avec les détails et l'adresse e-mail de votre employé via le lien suivant : info@yeatapp.com. Lorsque votre employé télécharge l'application, nous nous assurons qu'il/elle peut gérer votre restaurant avec son adresse e-mail.

YEAT vous informe sur toutes les tables de dernière minute publiées dans vos restaurants préférés. Choisissez vos favoris, activez vos notifications et c'est parti.

Lorsque votre restaurant préféré a une table libre, vous recevrez une notification. Réservez et Yes Eat. Yeat.
