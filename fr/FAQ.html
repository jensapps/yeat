Comment les restaurants sont-ils affichés sur Yeat?

Pour une utilisation optimale, il est important d'activer vos services de localisation sur votre téléphone. À l'ouverture de l'application, vous obtenez immédiatement un aperçu de tous les restaurants qui ont une table disponible dans votre quartier. Si vous avez sélectionné des favoris, vous pouvez également consulter cette liste. Si votre fonction de localisation est désactivée, vous pouvez uniquement consulter la liste des tableaux disponibles par ordre alphabétique.

Comment gérer les notifications?

vous pouvez définir vos préférences pour les notifications en fonction de vos besoins. Vous pouvez gérer les notifications sur votre téléphone à tout moment. Ils remplacent de toute façon les paramètres de l'application. Étant donné la nature de dernière minute de l'application, nous vous recommandons de laisser vos notifications pour Yeat afin de ne manquer aucune opportunité.

Voulez-vous obtenir une table pour l'un des prochains jours dans un restaurant généralement plein? puis marquez le ou les restaurants souhaités comme vos favoris, assurez-vous que vos notifications sont activées sur votre téléphone. Ensuite, vous serez informé de tous vos restaurants marqués comme favoris lorsqu'ils mettront une table en ligne.

En tant qu'utilisateur, dois-je payer quelque chose pour utiliser Yeat?

L'application est entièrement gratuite pour l'utilisateur. L'utilisateur accepte les termes et conditions de Yeat, en utilisant l'application. Lisez ici les conditions générales.

Que se passe-t-il lorsqu'une table réservée n'apparaît pas?

Les conditions générales, approuvées par l'utilisateur, stipulent qu'un utilisateur qui a réservé la table s'engage à prendre la table avec le nombre de personnes disponibles à la date et à l'heure spécifiées. Si l'utilisateur ne se présente pas sans aucune forme de communication, le restaurateur se réserve le droit de demander à l'utilisateur une compensation pour cela. Ceci est une décision du propriétaire du restaurant lui-même et se fait directement entre l'utilisateur et le propriétaire du restaurant.

Combien de tableaux puis-je mettre en ligne en même temps?

Autant que vous le souhaitez.

Combien de temps ma table reste-t-elle en ligne?

Lorsque vous placez votre table en ligne, vous devez indiquer jusqu'à quelle heure vous souhaitez rendre la table visible. Lorsque la table est réservée, l'utilisateur est informé que la table n'est plus disponible. Ce message reste pour une courte période.

Puis-je mettre une table hors ligne?

Vous pouvez mettre une table hors ligne à tout moment sans frais supplémentaires.

Comment les utilisateurs peuvent-ils voir mes restaurants?

Lorsque vous enregistrez votre restaurant pour inscription, vous entrez vous-même toutes vos coordonnées et vous pouvez également télécharger quelques photos. Les utilisateurs peuvent ensuite vous trouver, soit par nom, emplacement ou d'autres filtres tels que le score Gault & Millau ou les étoiles Michelin.

Je suis propriétaire d'un autre restaurant, dois-je créer un autre compte?

Ce n'est possible que pour 1 compte par adresse professionnelle.

Je veux qu'un employé gère Yeat. Que devrais-je faire ?

Envoyez-nous un e-mail avec les détails et l'adresse e-mail de votre employé via le lien suivants Link.. Lorsque votre employé télécharge l'application, nous nous assurons qu'il peut gérer votre restaurant avec son adresse e-mail.
